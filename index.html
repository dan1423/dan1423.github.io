Projects

NQueens Algorithm

Nqueens is a project designed with HTML5 canvas and javascript. The purpose of the project is to display how the 8-Queens puzzle works using backtracking algorithm. Viewer can select a board size. When there is a solution, a snapshot is saved at the bottom of the page.

Eventer

An android project developed with Android Studio, Eventer is a scheduling/notification app. Users can set multiple notifications, import and export notification with Google Sheets.The app utilizes Google Drive and Google Sheets API. It is available for download at the play store .
